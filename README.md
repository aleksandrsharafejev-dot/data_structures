# Data Structures Project

This project implements several fundamental data structures in Python.

Implemented structures:

* Linked List
* Circular Linked List
* Stack
* Queue
* Binary Search Tree
* Hash Table
* Max Heap
* Priority Queue
* Deque

The project also includes basic demonstrations and visualizations of some structures.

## Visualization

The Binary Search Tree and Linked List made by myself can be visualized with tree_visualization.py and linked_list_visualization.py.
Also the visualizations can be opened through the provided index.html file in the project.

## Technologies

* Python
* Matplotlib

## Purpose

The goal of this project is to practice core Computer Science concepts such as data structures, algorithms, and time complexity.
